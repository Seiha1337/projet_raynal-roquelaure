# 🏭 Projet BTS - Raynal & Roquelaure

Bienvenue sur le dépôt global de mon projet de BTS réalisé pour l'entreprise **Raynal & Roquelaure**. 
Ce dépôt centralise l'ensemble des codes, documentations et applications liés à la supervision et à l'automatisation d'un autoclave industriel.

---

## 📂 Architecture du Projet

Voici le contenu des différents modules de ce dépôt :

* 📁 **`Autoclave-Raynal-Roquelaure`** : Code source principal (C++ / ESP32) du simulateur d'autoclave avec régulation T.O.R, écran TFT et serveur Modbus TCP.
* 📁 **`APPLICATION`** : Fichiers liés à l'application web/IHM (Interface Homme-Machine) pour le contrôle à distance.
* 📁 **`Automate OPTA Finder`** : Programmes et logiques liés à l'automate industriel (PLC).
* 📁 **`testModbusRtuRS485_WT32`** : Codes de tests et maquettage pour la communication série RS485 (Modbus RTU).
* 📁 **`DOCUMENTATION`** : Rapports, schémas électriques, cahier des charges et ressources techniques du projet.
* 📁 **`Autoclave CODE ESP32`** : (Anciens codes ou versions de secours).

---

## 🛠️ Technologies Utilisées

* **Microcontrôleur :** ESP32 (WT32-ETH01)
* **Automate :** Finder OPTA
* **Protocoles :** Modbus TCP/IP, Modbus RTU (RS485)
* **Langages :** C/C++ (PlatformIO), HTML/CSS (Interface Web)
